Memory is not a unitary function but a set of related neural processes including working memory (short-term manipulation), episodic memory (events and experiences), semantic memory (facts and knowledge), and procedural memory (skills and habits). Each depends on distinct but overlapping neural systems: hippocampus for episodic memory formation, prefrontal cortex for working memory, cortical distributed networks for semantic storage, and basal ganglia/cerebellum for procedural memory.

Memory consolidation occurs primarily during sleep, with slow-wave sleep supporting declarative memory consolidation and REM supporting procedural and emotional memory. This makes sleep the single highest-yield intervention for memory function. Chronic sleep deprivation (less than 6 hours nightly) produces measurable episodic memory deficits that are difficult to compensate with any supplement. Physical exercise, particularly regular aerobic activity, increases hippocampal volume and BDNF production with effect sizes on memory performance exceeding any pharmaceutical or supplement intervention.

Supplementation evidence for memory is modest and often indication-specific. Creatine monohydrate (5g daily) improves memory performance specifically under sleep-deprived or stressed conditions, with larger effect in vegetarians (who have lower baseline muscle creatine stores). Bacopa monnieri (300mg standardized extract) improves memory consolidation over 8-12 weeks of daily use. Lion's mane shows preclinical BDNF/NGF elevation with mixed clinical data. For age-related cognitive decline, phosphatidylserine (100mg three times daily) has modest evidence in older adults with subjective cognitive complaints. None of these interventions substitute for foundation interventions (sleep, exercise, Mediterranean diet, social engagement, cognitive activity) that consistently produce larger effects.
